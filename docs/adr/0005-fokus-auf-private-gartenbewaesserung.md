# Fokus auf private Gartenbewässerung

Irrigation Manager wird für die Wasserbewässerung privater Gärten entwickelt. Professionelle Landwirtschaft, Gewächshausklimatisierung, Ertragsmodelle sowie Dünger- oder Chemikaliendosierung bleiben ausdrücklich außerhalb des Umfangs, damit Sicherheitsmodell und Benutzeroberfläche auf den Hausgarten fokussiert bleiben.
