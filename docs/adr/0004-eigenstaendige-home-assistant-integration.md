# Eigenständige Home-Assistant-Integration entwickeln

Irrigation Manager wird als eigenständige UI-konfigurierbare Home-Assistant-Custom-Integration entwickelt. Reine Automationen sowie Kombinationen aus Irrigation Unlimited, Smart Irrigation oder NeverDry wurden verworfen, weil die gemeinsame Verantwortung für Planung, Wasserbilanz, Verbrauchszuordnung und sichere Ventilsteuerung sonst über mehrere Konfigurationen und Zustandsmodelle verteilt bliebe.
