# Ventile exklusiv steuern

Irrigation Manager besitzt die exklusive Steuerhoheit über die zugeordneten Haupt- und Zonenventile. Eine außerhalb der Integration erkannte Ventilöffnung wird geschlossen und löst eine Sicherheitssperre aus, statt als manuelle Bewässerung übernommen zu werden; dadurch werden konkurrierende Automationen und nicht nachvollziehbare Wasserabgaben vermieden.
