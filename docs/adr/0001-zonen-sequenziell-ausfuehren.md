# Zonen sequenziell ausführen

Innerhalb einer Bewässerungsanlage darf immer nur eine Zone gleichzeitig bewässert werden. Dadurch kann der gemeinsame Wasserzähler den tatsächlichen Verbrauch eindeutig einer Zone zuordnen; zugleich bleiben Hydraulik und Fehlerbehandlung einfacher, auch wenn Parallelbetrieb die Gesamtdauer verkürzen könnte.
